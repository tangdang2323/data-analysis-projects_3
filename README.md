# Анализ платежей клиентов фитнес-клубов

**Заказчик:** Сеть фитнес-клубов, работающих в разных городах России. Клиенты могут посещать индивидуальные или групповые тренировки. Каждое занятие оплачивается заранее, перед его началом.

**Задача:**  
Проанализировать платежи клиентов, рассчитать средние показатели, выявить аномалии и подготовить данные для дальнейшего моделирования.

## Цель
Провести первичный анализ платежей, очистить данные от выбросов и подготовить витрину для бизнес-отчётов.

## Инструменты
- Python (pandas, numpy, matplotlib, seaborn)
- Jupyter Notebook

## Этапы работы

### 1. Загрузка и первичный анализ данных
- Изучена структура таблиц и проведен разведочный анализ данных
- Проверены типы данных, пропуски, дубликаты.

### 2. Расчёт среднего платежа и поиск выбросов (метод IQR)
- Рассчитан межквартильный размах (IQR)
- Определены границы: `Q1 - 1.5 * IQR` и `Q3 + 1.5 * IQR`
- Все платежи за пределами этих границ помечены как выбросы
- Рассчитан средний платеж для всех пользователей после исключения выбросов
- Найдено среднее значение для каждого месяца
  
[Смотреть Jupyter Notebook](fitness_payment_analysis.ipynb)

### 3. Анализ возраста посетителей 
- Проанализирован список возрастов клиентов фитнес-клубов
- Рассчитан средний возраст всех посетителей
- Написана функция для расчёта среднего возраста с пороговым значением
- Рассчитан межквартильный размах (IQR), определены границы выбросов
- Исключены возрастные выбросы и рассчитано скорректированное среднее
- Написана функция для автоматического исключения выбросов и расчёта среднего возраста
  
[Смотреть Jupyter Notebook](fitness_age_analysis.ipynb)

### 4. Анализ влияния маркетинговой акции 
- Данные разделены по пользователю и месцу
- Рассчитана выручка и количество тренировок в динамике по месяцам
- Рассчитаны LT и LTR
- Проведена оценка эффекта от маркетинговой акции
  
[Смотреть Jupyter Notebook](marketing_analysis.ipynb)

### 4. Анализ общей информации о пользователях 
- Проведен разведоточный анализ данных для таблицы, содержащей информацию о пользователях
- Рассчитан межквартильный размах (IQR), определены границы выбросов
- Построено распределение пользователей по полу и городам
- Рассчитано среднее количество тренировок в месяц для каждого пользователя
  
[Смотреть Jupyter Notebook](users_info_analysis.ipynb)

## Результаты
- Средний платёж по сети составил 3645.71 рублей
- Средний возраст посетителей по сети составил 30 лет
- LTR 36000 рублей, LT 7 мес
- Прирост выручки из-за проведения акции 390800
- Прирост прибыли из-за проведения акции -400
